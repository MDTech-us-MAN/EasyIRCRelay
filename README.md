# EasyIRCRelay
EasyIRCRelay is a simple IRC bot that relays messages between channels on different servers.

# Requirements
 * QT 5 (with qmake)
	It can be easily installed on ubuntu using `sudo apt install qt5-default`.

# Installation

1. `qmake`
2. `make`
3. Edit the `config.json` file.
4. `./EasyIRCRelay`
